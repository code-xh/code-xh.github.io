软件的安装与准备

#1、Git 的安装

Git是用来管理代码的，在 Windows 下用 cmd 貌似也可以实现，但比较麻烦，所以推荐下载此软件。

Git官网

缺省安装即可（即一路 next）。
安装成功后，随意在一个文件夹中右键，会出现 Git Bash Here

#2、Node.js 的安装

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境。使用 hexo 需要安装此环境。

Nodejs官网

选择左边的 LTS 版本，该版本更稳定。缺省安装即可。
